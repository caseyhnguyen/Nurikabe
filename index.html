Nurikabe

Nurikabe is a logic based game played on a grid of cells, some of which are numbered. Cells which aren't numbered can either be white or black. The player is tasked with changing the color of the non-numbered cells so that the following rules are met.

  1. Each numbered cell is in an island, and the number of cells in each island corresponds to the numbered cell in that island.
  2. Each island must contain exactly one numbered cell.
  3. There must be only one sea, which can't contain any pools.

Definitions:
Adjacent - Horizontally or vertically, but not diagonally.
Island - A collection of white cells adjcent to each other, including the numbered cell.
Sea - A collection of black cells adjacent to each other.
Pool - A 2x2 collection of black cells.
